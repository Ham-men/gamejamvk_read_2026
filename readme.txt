ОСТРОВ ПОТЕРЯННЫХ МЕЛОДИЙ
====================================

КАК ЗАПУСТИТЬ ПРОЕКТ:
--------------------
1. Откройте папку проекта: ...game_v2
2. Запустите локальный веб-сервер (один из способов):

   Способ А (Python):
   - Откройте командную строку (cmd)
   - Перейдите в папку: cd ...game_v2
   - Запустите: python -m http.server 8000

   Способ Б (Node.js):
   - Установите: npm install -g http-server
   - Запустите: http-server -p 8000

   Способ В (PHP):
   - Запустите: php -S localhost:8000

3. Откройте браузер (рекомендуется Chrome для распознавания речи)
4. Перейдите по адресу: http://localhost:8000

ВАЖНО: Если запускать просто как файл (file://), браузер будет каждый раз спрашивать разрешение на микрофон. Через локальный сервер разрешение запоминается!

ЗВУК: При переходе между частями сказки проигрывается sound.mp3 (добавлено)

ОПИСАНИЕ СКАЗОК (УРОВНИ):
------------------------

1. ЖЕНИХ
   Тип: Чтение
   Сюжет: История о Наташе, которая пропадала три дня, и её родителях, пытающихся узнать тайну.
   Что делать: Читайте текст вслух, произносите ключевые слова (Наташа, отец, мать). После упоминания слов на сцене появляются спрайты (эмодзи).

2. ПЕСНЬ О ВЕЩЕМ ОЛЕГЕ
   Тип: Чтение
   Сюжет: Легендарный князь Олег вещий отправляется в поход против хазар.
   Что делать: Читайте текст Пушкина вслух. Ключевые слова: Олег, хазарам, князь.

3. СКАЗКА О ЗОЛОТОМ ПЕТУШКЕ
   Тип: Чтение
   Сюжет: Царь Дадон стареет и хочет отдохнуть от ратных дел, но соседи не дают покоя.
   Что делать: Читайте сказку Пушкина. Ключевые слова: царь, Дадон, соседям.

4. СКАЗКА О МЕДВЕДИХЕ
   Тип: Чтение
   Сюжет: Медведиха с медвежатами гуляет в лесу, появляется мужик с рогатиной.
   Что делать: Читайте текст вслух. Ключевые слова: лесу, медведиха, мужик.

5. СКАЗКА О ПОПЕ И О РАБОТНИКЕ ЕГО БАЛДЕ
   Тип: Чтение
   Сюжет: Поп нанимает работника Балду за три щелчка по лбу.
   Что делать: Читайте сказку Пушкина. Ключевые слова: поп, Балда, лоб.

6. СКАЗКА О РЫБАКЕ И РЫБКЕ
   Тип: Чтение
   Сюжет: Старик ловит золотую рыбку, которая обещает исполнить любое желание.
   Что делать: Читайте сказку Пушкина. Ключевые слова: старик, старухой, моря.

7. У ЛУКОМОРЬЯ ДУБ ЗЕЛЕНЫЙ
   Тип: Чтение
   Сюжет: Знаменитые строки из поэмы Пушкина "Руслан и Людмила".
   Что делать: Читайте текст вслух. Ключевые слова: дуб, кот, лес.

8. ОСТРОВ ПОТЕРЯННЫХ МЕЛОДИЙ (ОСНОВНОЙ СЮЖЕТ)
   Тип: Смешанный (чтение + пение)
   Сюжет: Сверчок Чик путешествует по острову, возвращая музыку и сказки.
   
   Части уровня:
   - Чтение: Знакомство с островом (ключевые слова: Чик, Тишиниус, остров)
   - Пение: Спойте ноту "До" чтобы разбудить Чика
   - Чтение: Деревня Ноток (ключевые слова: Чик, теремок, деревня)
   - Пение: Спойте ноту "Ре" чтобы разбудить птичку
   - Чтение: Сухое русло реки (ключевые слова: речку, дождик, мост)
   - Пение: Спойте ноту "Ми" чтобы починить мост
   - Чтение: Пещера Шёпота (ключевые слова: пещера, нотка, кристаллы)
   - Пение: Спойте ноту "Фа" чтобы зажечь кристаллы
   - Чтение: Битва с Тишиниусом (ключевые слова: Тишиниус, музыка, остров)
   - Пение: Гимн острова - спойте ноты "До, Ре, Ми, Соль"

------------------------------------
УПРАВЛЕНИЕ:
--------------------
- Персонаж (Чик) движется автоматически при чтении текста
- Нажмите "Начать" чтобы начать чтение вслух
- Нажмите "Начать петь" чтобы начать пение в микрофон
- После завершения части нажмите "Дальше" или "Стоп"


------------------------------------
СОВЕТЫ:
--------------------
- Используйте браузер Chrome (лучшая поддержка SpeechRecognition)
- Разрешите доступ к микрофону при первом запросе
- Говорите четко и громко при чтении
- Ноты пойте чисто, близко к микрофону
- Если слово не зачеркивается - проверьте, правильно ли оно написано в тексте
